# 🧮 Calculator Project

## 📌 Description
This is a simple calculator built using Python and Tkinter.  
It can perform basic arithmetic operations like addition, subtraction, multiplication, and division.

## 🚀 Features
- User-friendly GUI
- Basic operations (+, -, *, /)
- Clear button
- Error handling

## 🛠️ Tech Used
- Python
- Tkinter (GUI library)

## ▶️ How to Run
1. Install Python
2. Download the project
3. Run this command:

python calculator.py
